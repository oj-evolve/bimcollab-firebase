Frequently Asked Questions
How do I reset my project password?
Currently, project keys are managed by the project owner. If you have lost your access key, please contact the project owner directly. For security reasons, we do not store raw passwords on our servers.
What file formats are supported?
BIM COLLAB supports a variety of formats including:
  • Images: JPG, PNG, WEBP, SVG, BMP
  • Documents: PDF (viewing), DOCX, XLSX, PPTX, CSV (download only)
  • 3D Models: GLB, GLTF
  • CAD: DWG, DXF (download only)
  • Media: MP4, WEBM, MP3, WAV
How do I invite team members?
Team members can join by signing up with their email or Google account. Once logged in, they can select their professional role. To access a specific project, they may need the Project ID provided by the Project Owner.
Is my data secure?
Yes. We use end-to-end encryption for chat messages. Your project files are stored securely in the cloud, and access is restricted to users with the correct project credentials.
Can I use this offline?
The application has basic offline capabilities. You can view previously loaded content, but real-time chat and new file uploads require an active internet connection. Changes made offline will sync when you reconnect.
